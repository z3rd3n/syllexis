### 2. Welche Daten die App verarbeitet

Syllexis verarbeitet deine Lerndaten ueberwiegend lokal auf deinem Geraet. Dazu gehoeren insbesondere selbst angelegte Decks, hinzugefuegte Woerter, Lernfortschritt, Trainingseinheiten, Tagesstatistiken und App-Einstellungen.

Wenn du eine Anki-Datei importierst oder ein Deck exportierst, verarbeitet die App nur die von dir ausgewaehlten Dateien lokal auf deinem Geraet. Exportierte Dateien werden nur gespeichert oder geteilt, wenn du das selbst ausloest.

Die App greift nicht von sich aus auf Kontakte, Standort, Kamera oder Mikrofon zu.
